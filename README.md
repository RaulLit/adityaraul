### 🧪 Setup Instructions

```

git clone https://github.com/adityaraul/portfolio.git 
cd portfolio 

# Install dependencies 
npm install 

# Start the dev server 
npm run dev
```
> ⚠ Make sure Node.js ≥ 18 is installed.
